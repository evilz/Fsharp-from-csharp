# Fsharp-from-csharp

Petite démo d'interopérabilité entre une application C# et une librairie F#.

## Prérequis

- .NET SDK (version compatible avec la solution)

## Démarrer

```bash
# Restaurer les dépendances

dotnet restore

# Compiler la solution

dotnet build

# Exécuter l'app C#

dotnet run --project CsharpApp
```
